# oop-capsulation-002
An encapsulation exercise for oop practice

Build a Student class
Apply encapsulation to model a university student. All fields must be private with proper validation in every setter.
Requirements
1
Field name (String) — cannot be null or blank
2
Field grade (double) — must be between 0.0 and 100.0 inclusive
3
Field studentId (String) — set once in the constructor, no setter (read-only)
4
Method getLetterGrade() — returns "A" (90+), "B" (75+), "C" (60+), or "F" (below 60)
5
Override toString() — returns something like Student{id='S001', name='Alice', grade=88.5, letter='B'}
